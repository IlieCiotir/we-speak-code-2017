#CSS examples
Spinning an element when the mouse hovers over it 
.rotate-on-hover {
    text-align: center;
    font-size: 20px;
    border: 1px solid #eee;
    width: 100px;
    padding: 20px;
    margin-right: auto;
    margin-left: auto;
}

.rotate-on-hover:hover {
    animation: rotate-on-hover-animation;
    animation-direction: reverse;
    animation-duration: 1s;
    animation-direction: alternate;
    animation-iteration-count: 3;
    transform: rotate(190deg);
}

@keyframes rotate-on-hover-animation {
    from {
        transform: rotate(0deg);
    }
    to {
        transform: rotate(190deg);
    }
}
